# 🤖 Model Advisor

**Find the perfect AI model for your needs in seconds.**

A completely free, open-source tool that helps you compare AI models by cost, performance, quality, and more. Get unbiased recommendations based on your specific use case.

## ✨ Features

✅ **Intelligent Recommendations** - Get ranked model suggestions based on your needs
✅ **Cost Calculations** - See estimated monthly costs for different usage scenarios
✅ **Community Insights** - Real discussions from Reddit about each model
✅ **Unbiased** - Recommendations are objective, not vendor-biased
✅ **Free to Use** - No sign-up, no payment required
✅ **Easy to Share** - Copy, print, or share your recommendations

## 🎯 How It Works

1. **Tell us your needs** - Use case, volume, budget, priorities
2. **We research** - Search latest models, Reddit discussions, community feedback
3. **You get recommendations** - Top 10+ models ranked by fit to your needs
4. **See the costs** - Breakdown for light/medium/heavy usage
5. **Choose wisely** - Make an informed decision

## 💻 Deployment

### Quick Start (20 minutes)

1. Get free API keys:
   - Groq: https://console.groq.com
   - Reddit: https://www.reddit.com/prefs/apps

2. Create `.env` file:
   ```
   GROQ_API_KEY=your_key_here
   REDDIT_CLIENT_ID=your_id_here
   REDDIT_CLIENT_SECRET=your_secret_here
   REDDIT_USER_AGENT=model-recommender-bot/1.0
   ```

3. Push to GitHub and deploy to Netlify

See **DEPLOYMENT_GUIDE.md** for detailed instructions.
